Title: Simulation And Analysis For Real And Reactive Power Control With Series Type FACTS's Device

Description: A thyrister controlled series capacitor (TCSC) belongs to the flexible alternating current transmission system (FACT's) family of devices. It is a variable capacitive an inductive reactance device that can be used to provide series compensation in power transmission line. The considered topology of TCSC consists of back to back thyrister that can control the reactance of TCSC. By changing the firing angle of this back to back thyrister it is possible to vary the reactance of the TCSC. This Simulation is carried out by using software package MATLAB Simulink.
